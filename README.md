# Projeto-sistema-bancario-python
Projeto na qual desenvolvi pelo curso da Dio.me sobre python. Ond construímos um sismples sistema bancário com depósito, saque, consultar extrato e sair do sistema.

O sistema poderá realizar depósitos usando apenas números inteiros positivos.
O sistema poderá realizar 3 saques diários, com um limite de valor de R$500.
O sistema irá informar no extrato todas as movimentações feitas no sistema naquele dia.
